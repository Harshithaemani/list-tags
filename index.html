Types of Lists

What is HTML?
  • HTML stands for Hyper-text Markup Language.
  • HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page.
  • HTML consists of a series of elements.
  • HTML elements tell the browser how to display the content.
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

  3. h3 - This is the 3rd heading.
  4. h4 - This is the 4th heading.
  5. h5 - This is the 5th heading.
  6. h6 - This is the 6th heading.
Paragraph Tag
  • The "p" tag in HTML signifies a paragraph.
  • As a block-level element, a new paragraph inherently starts on a fresh line, with browsers intuitively adding space before and after a paragraph for enhanced readability.
  • If you put a lot of spaces inside the HTML p tag, browser removes extra spaces and extra line while displaying the page.
  • The browser counts number of spaces and lines as a single one.

List Tags

These are classified into 3 Types

  1. Ordered list tags
  2. Unordered list tags
  3. Description list

1. Ordered list tags:
  • It is used to specify list items in sequence manner
  • It is defined with "ol" tag
  • By default it shows numbered order

Ordered list Attributes

  • type- It is used to specify the type of list itmes (A, a, i, I, 1)
  • start- from where the list to be started
  • reversed- number order will be reversed

2. Unordered list tags
  • It is used to specify list items in particular order
  • It is defined with "ul" and list items are defined "li" tags
  • By default it shows with bullet list

Unordered list Attributes

  • type- cirle, disc, square

3. Description List
  • It is used to create description items in html document
  • It is defined with "dl" tag
  • It has description term and description data
  • Description term is defined with dt & Description data is defined with dd
